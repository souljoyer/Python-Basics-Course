## 📋 Current Project: CLI Task Manager

A command-line interface task manager application that demonstrates core Python programming concepts and best practices.

### ✨ Key Features

- **CRUD Operations**: Create, Read, Update, and Delete tasks with ease
  - Add new tasks
  - Edit existing tasks
  - Mark tasks as complete
- **Data Persistence**: Automatic save and load functionality using local text file storage
- **Error Handling**: Robust input validation and error management for a smooth user experience
- **User-Friendly Interface**: Intuitive command-line prompts and clear feedback

### 🛠️ Technologies Used

- Python 3.x
- File I/O operations
- Data structures (lists, dictionaries)
- Input validation and error handling

### 🚀 How to Use

```bash
# Run the task manager
python task_manager.py
```

Follow the on-screen prompts to:
- Add new tasks
- View all tasks
- Edit task details
- Mark tasks as complete
- Delete tasks
